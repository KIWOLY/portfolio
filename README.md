🌐 Personal Portfolio – React & Tailwind CSS

This is my personal portfolio website built using React, Tailwind CSS, and Vite.
It highlights my skills, projects, and background with a modern, clean, and responsive design.



## Tech Stack

Frontend: React

Styling: Tailwind CSS

Build Tool: Vite

Language: JavaScript (ES6+)

Linting: ESLint

Version Control: Git & GitHub

## Features

⚛️ Built with modern React

🎨 Styled using Tailwind CSS

📱 Fully responsive design

⚡ Fast development and build using Vite

🧩 Clean and simple folder structure

🔧 ESLint configured for code quality

🛠️ Installation & Setup

Follow these steps to run the project locally:

1️⃣ Clone the repository
git clone https://github.com/KIWOLY/portfolio.git

2️⃣ Navigate into the project directory
cd portfolio

3️⃣ Install dependencies
npm install

4️⃣ Start the development server
npm run dev


The project will run at:

http://localhost:5173

🎨 Tailwind CSS Usage

Tailwind CSS is used for styling components using utility classes.

Example:

<div className="bg-background text-foreground p-6 rounded-xl">
  Welcome to my portfolio
</div>
